JavaScript

JavaScript е објектно-ориентиран скриптирачки јазик кој овозможува имплементација на сложени функции на веб-страници.

JavaScript овозможува создавање динамично ажурирана содржина, контрола на мултимедија, анимирање слики и многу други работи кој што ја прават веб-страницата подобра.

JavaScript се применува на HTML страницата на сличен начин како CSS. JavaScript се повикува во HTML со тагот < script >.

Кодот може да се пишува во рамките на телото на тагот, или може да се повика надворешен фајл со имплементирање на src во рамките на script тагот.
