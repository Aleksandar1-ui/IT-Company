Bootstrap

Bootstrap е HTML, CSS и JS библиотека која се фокусира на поедноставување на развојот на информативни веб-страници.

Примарната цел на негово додавање на веб-проект е да се применат изборите на Bootstrap за боја, големина, фонт и распоред на тој проект. Откако ќе се додаде во проект, Bootstrap обезбедува основни дефиниции за стилови за сите HTML елементи.

Покрај тоа, програмерите можат да ги искористат предностите на класите CSS дефинирани во Bootstrap за дополнително да го приспособат изгледот на нивната содржина.

Bootstrap доаѓа и со неколку JavaScript компоненти за кои не се потребни други библиотеки како jQuery. Тие обезбедуваат дополнителни елементи на корисничкиот интерфејс како што се dialog boxes, совети за алатки, ленти за напредок, паѓачки мени за навигација и вртелешки.
