JSON

JSON, или JavaScript Object Notation, е отворен стандард базиран на текст, дизајниран за размена на податоци разбирлива за луѓето.

Изведен е од јазикот JavaScript за претставување едноставни структури на податоци и асоцијативни низи, наречени објекти.

И покрај неговата поврзаност со JavaScript, тој е независен од јазикот.Форматот JSON често се користи за серијализирање и пренос на структурирани податоци преку мрежна врска.

Тој првенствено се користи за размена на податоци помеѓу серверот и веб апликацијата, како замена за XML.
